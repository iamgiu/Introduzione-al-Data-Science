# Introduzione al Data Science

> Un percorso pratico di apprendimento al Data Science attraverso Jupyter Notebook, organizzato settimana per settimana.

---

## Descrizione

Questo repository raccoglie materiale didattico e notebook interattivi pensati per chi si avvicina per la prima volta al mondo del **Data Science**. Il contenuto è strutturato in moduli settimanali progressivi, che guidano lo studente dai concetti fondamentali fino alle tecniche di analisi e manipolazione dei dati.

Ogni settimana introduce nuovi strumenti e metodologie, con esempi pratici e codice eseguibile direttamente in ambiente Jupyter.

---

### Prima Settimana
Introduzione agli strumenti di base del Data Science:
- Ambiente di lavoro con Jupyter Notebook
- Primi passi con Python per l'analisi dei dati
- Librerie fondamentali (NumPy, Pandas, Matplotlib)
- Esplorazione e visualizzazione di dataset
